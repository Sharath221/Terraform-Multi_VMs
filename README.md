# terraform-aws and azure 

terraform-aws
Check variables in ```hcl input.tfvars ``` apply the required count

```hcl
Commands
# terraform init
# terraform plan --var-file input.tfvars
# terraform apply --var-file input.tfvars
# terraform destroy --var-file input.tfvars
```
terraform-azure

Commands
